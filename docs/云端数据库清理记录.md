## 6. 防止再次污染

`PlanningServiceTest` 和 `WarehouseProductionServiceTest` 会写数据库。现在检测到远程数据库时默认不执行这些测试方法，普通 `mvn test` 和 `mvn package` 不再向云库写测试数据。

确需对远程数据库执行破坏性集成测试时，必须显式增加：

```powershell
-Dmes.allowDestructiveRemoteDbTests=true
```

该开关只能用于明确准备好的测试数据库，不应在当前云端验收库上使用。
