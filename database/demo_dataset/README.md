# MES 轮胎追溯 3 万条演示数据集

文件：`mes_tire_trace_30000.csv`

规模：30,000 行，每行对应一条轮胎/一个二维码追溯记录。

用途：
- 二维码追溯演示：扫码后按 `trace_code` 或 `qr_code_value` 查询单胎全流程。
- 质量分析：可按 `inspection_result`、`defect_desc`、`rfv_n`、`lfv_n`、`dynamic_balance_g` 做异常统计。
- 大数据/AI 展示：可按产品、批次、设备、产线、客户、时间做趋势分析。
- MES 流程串联：订单、工单、批次、物料批次、设备、质检、状态都在同一行里。

说明：
- 产品品牌和规格参考公开轮胎行业命名方式。
- 客户、批次、工单、质检数值为模拟数据，不是真实企业生产数据。
- `mes_tire_trace_30000_table.sql` 只负责建一张独立演示表，不会覆盖现有业务表。
